Algumas observações:
-Nessa implementação as posições do Labirinto começa no 1 em vez do tradicional 0.

-Requisitos:
Ter instalado python
Ter instalado uma IDE para visualizar o código

-Implementaçõe:
Busca BFS(busca em lagura)
Busca A*

-Como compilar:
1-) Com o promt aberto na pasta onde arquivos estão presentes
2-) Cole o seguinte comando "python AstarBFS.py"
3-) insira o nome do arquivo teste.txt ou outro arquivo no mesmo formato
4-) Depois insira o número de linhas pressione Enter e depois o número de colunas
5-) Coloque as coordenadas inicias do Labirinto no caso do exemplo é 1 3 e o final é 29 23
6-) Após mostra na tela escolha o método de busca.
